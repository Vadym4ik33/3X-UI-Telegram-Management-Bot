# 3X-UI-Telegram-Management-Bot
Telegram-бот для удобного администрирования VPN-сервера на базе панели **3X-UI (Xray)**.   Скрипт автоматизирует рутинные задачи администратора: создание пользователей, продление подписок, подсчет потраченного трафика и мониторинг состояния сервера прямо из Telegram.

## ✨ Основные возможности

* **📊 Мониторинг сервера:** Загрузка ЦП (CPU), оперативной памяти (RAM) и проверка статуса процесса `xray`.
* **👥 Управление пользователями:**
  * Генерация новых клиентов с привязкой к нескольким Inbound-подключениям.
  * Создание ссылок на подписки (`sub/`) и готовых конфигураций (`vless://`, `trojan://` и др.).
  * Настройка гибких лимитов по времени (в днях или безлимит).
  * Продление подписок (автоматически суммирует оставшиеся дни).
  * Быстрое удаление пользователей по email.
* **📈 Учет трафика:** Автоматический сбор актуальной статистики входящего/исходящего трафика по всем инбаундам с понятным форматированием (`KB`, `MB`, `GB`).
* **🔒 Сессии и авторизация:** Поддержка получения CSRF-токенов и автоматический повторный вход (`re-login`), если сессия с панелью прервалась.
* **🛡 Защита от переполнения:** Автоматическая разбивка длинных списков пользователей на отдельные сообщения (обход лимита Telegram в 4096 символов).

---

## 🛠 Технологический стек

* **Language:** Python 3.8+
* **Libraries:** `pyTelegramBotAPI` (telebot), `requests`, `psutil`, `urllib3`
* **Target Panel:** 3X-UI / Xray-core API

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/Vadym4ik33/3X-UI-Telegram-Management-Bot.git
cd 3X-UI-Telegram-Management-Bot
```
### 2. Установка зависимостей
```
pip install pyTelegramBotAPI requests psutil urllib3
```
### 3. Настройка конфигурации
```nano bot.py
BOT_TOKEN = "ВАШ_ТОКЕН_БОТА"
ADMIN_CHAT_ID = 123456789  # Ваш Telegram ID

PANEL_BASE_URL = "https://127.0.0.1:52313/subpath"  # Адрес панели
PANEL_USERNAME = "admin"                            # Логин
PANEL_PASSWORD = "your_password"                    # Пароль

EXTERNAL_HOST = "your-domain-or-ip.com"            # Внешний хост для ссылок
EXTERNAL_PORT = 52313
INBOUND_IDS = [1, 2, 3]                            # ID инбаундов для привязки
```
### 4. Запуск бота
`python3 bot.py`
